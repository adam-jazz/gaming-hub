Gaming Hub Website

========================================

Overview:
This project is a multi-page gaming community website built using HTML, CSS, and JavaScript. It allows users to register, log in, view leaderboards, and simulate gameplay to earn points.

========================================

How to Run the Project:

1. Extract the ZIP file (if compressed).
2. Open the project folder.
3. Double-click on "index.html" to open the website in a browser.
4. Make sure the "images" folder is in the same directory as the HTML files.

========================================

Main Features:

* User Registration and Login system (using LocalStorage)
* User Profile with game statistics (wins, losses, score)
* Game selection page with interactive cards
* Individual leaderboard for each game
* Global leaderboard with ranking system
* "Play Game" simulation that updates user stats
* Dark mode toggle
* Responsive design for different screen sizes

========================================

How to Use:

1. Go to "Account" page.
2. Register a new account.
3. Login with your credentials.
4. Navigate to "Games" and select a game.
5. Click "Play Now" to simulate a match.
6. View your updated score in:

   * Game leaderboard
   * Global leaderboard
   * Profile dashboard

========================================

Notes:

* No external database is used.
* LocalStorage is used to simulate data storage.
* Data is saved in the browser and will persist unless cleared manually.

========================================

Project Structure:

* index.html → Home page
* games.html → Game selection page
* game-details.html → Individual game leaderboard
* leaderboard.html → Global leaderboard
* contact-us.html → Contact form
* account.html → Login/Register + Profile
* styles.css → Website styling
* script.js → All JavaScript logic
* images/ → Contains all images used
